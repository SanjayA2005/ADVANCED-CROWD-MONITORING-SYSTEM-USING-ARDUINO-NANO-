ADVANCED CROWD MONITORING SYSTEM USING ARDUINO NANO

OVERVIEW
The Advanced Crowd Monitoring System using Arduino Nano is an embedded solution designed to monitor and control crowd density in real-time. It helps prevent overcrowding in public places such as malls, classrooms, offices, and event venues by tracking the number of people entering and exiting a defined area. The system ensures safety, efficiency, and better space utilization.

OBJECTIVE
To monitor crowd density in real-time
To prevent overcrowding and ensure safety
To automate entry and exit tracking
To provide accurate occupancy data

TECHNOLOGIES USED
Arduino Nano
Embedded C / Arduino IDE
IR Sensors or Ultrasonic Sensors
LCD Display (16x2)
Buzzer and LED Indicators
ESP8266 (optional for IoT integration)

WORKING PRINCIPLE
Sensors are placed at the entry and exit points to detect movement. Each time a person crosses the sensor, the system updates the count accordingly. The Arduino Nano processes this data and maintains the current occupancy count. The value is displayed on the LCD screen. When the number of people exceeds the predefined limit, the system activates a buzzer or LED alert to indicate overcrowding.

FEATURES
Real-time people counting
Automatic entry and exit tracking
Overcrowding alert system
Compact and cost-effective design
Easy installation and maintenance
Expandable with IoT features

COMPONENTS REQUIRED
Arduino Nano
IR Sensor Modules (2)
LCD Display
Buzzer

OUTPUT
The system displays the current number of people inside the area on the LCD screen. When the limit is exceeded, a visual or audio alert is triggered to notify users.
